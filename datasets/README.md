# Sample datasets for testing the AI Data Engineer Agent

## Customer Churn Dataset
- **File**: customer_churn.csv
- **Rows**: 10,000
- **Columns**: 20
- **Target**: Churn (Yes/No)
- **Description**: Telecom customer churn prediction dataset

## House Prices Dataset
- **File**: house_prices.csv
- **Rows**: 1,460
- **Columns**: 81
- **Target**: SalePrice
- **Description**: Real estate price prediction dataset

## Sales Forecasting Dataset
- **File**: sales_data.csv
- **Rows**: 50,000
- **Columns**: 15
- **Target**: Sales
- **Description**: Retail sales forecasting dataset

## Credit Risk Dataset
- **File**: credit_risk.csv
- **Rows**: 30,000
- **Columns**: 25
- **Target**: Default
- **Description**: Credit risk assessment dataset

## Usage
Place these datasets in the `datasets/` folder for testing the AI Data Engineer Agent.

Each dataset will be automatically analyzed, cleaned, and used to train ML models.
